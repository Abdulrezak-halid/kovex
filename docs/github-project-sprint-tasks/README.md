# Kovex ERP - GitHub Project Sprint Tasks

This folder is the planning source for GitHub Project tasks. Each task can be
copied into GitHub Issues or GitHub Project cards.

## Files

- `completed-and-active-tasks.md`: existing project sprint tasks and completed
  work.
- `restructuring-and-cleanup-tasks.md`: project cleanup, file framework, rules,
  and professionalization tasks.

## Status Values

- Completed
- Ready
- In Progress
- Blocked
- Future

## Task Maintenance Rules

- Keep project rules in `PROJECT_RULES.md`.
- Keep sprint/project tasks in this folder.
- When a task is finished, update its checkbox from `[ ]` to `[x]`.
- Move completed tasks to the completed section or mark status as Completed.
- Add a short completion note after finishing a task.
